### Repetição Simples de WHILE com Estrutura de Repetição no início

					 var c = 1
					while (c <= 3) {                
						console.log(`REPETIR ${c}`)
						c += 1
					}

### Repetição de WHILE com DO com Estrutura de Repetição de teste Lógico no final

						  var c = 1
						  do {
								  console.log(`REPETIR ${c}`)
								  c++
						  } while (c <= 3)
